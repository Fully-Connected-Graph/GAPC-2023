Registration

This contest is open to everyone. Before registering, please read the rules of the contest and these can be found here.

In the contest, the teams can have 2 or 3 members. There is also the option to participate alone or to register alone and then be added in a random team. Both options can be requested in the registration form. Regarding teams, only the captain needs to make the registration and it can be done here.

Contact
